# OnePlayerSleep

Multi-version Minecraft plugin (1.16 - 1.21+) that skips night when any player enters a bed

## Build

```bash
mvn clean package
```

Output: `target/OnePlayerSleep-1.0.0.jar`

## Install

Place the JAR in your server's `plugins/` folder and restart.

## Compatibility

- Minecraft 1.16.x - 1.21.11+
- Paper/Spigot/Purpur servers
- Safe reflection-based version handling
