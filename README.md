# OnePlayerSleep

Multi-version Minecraft plugin (1.16 - 1.21+) that skips night when any player enters a bed

## Build

```bash
mvn clean package
```

Output: `builds/OnePlayerSleep-4.0.1.jar`

## Install

Place the JAR in your server's `plugins/` folder and restart.

## Compatibility

- Paper 1.16+
- Folia 1.21+
- Lightweight reflection-based scheduling to keep the hot path simple
