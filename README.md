A lightweight Minecraft Paper plugin that allows a single player to skip the night without requiring everyone on the server to sleep.

## Features

- One player can skip the night
- Optional weather clearing
- Fully customizable messages
- Lightweight and optimized
- Supports Minecraft 1.16–1.21+

## Compatibility

- Paper
- Purpur
- Spigot

## Performance

Designed to have a negligible impact on server performance.
